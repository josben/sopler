About Sopler.

What is Sopler ?

Sopler is a free/open source, social and collaborative web application for making to-do lists. Create them, share them and collaborate with your friends to organize your everyday life.


Sopler is based on three principles:

  • /. Simplicity - Ease of use for everyday people.
  • /. Security - Private data must remain private.
  • /. Sociability - Share and collaborate with those you choose, efficiently and fast.

The main idea behind Sopler.

We wanted the means to organize our everyday life. What we needed was to do it fast, socially and in collaboration with others. All these without any time-wasting registration and with respect to our privacy.

Your privacy and anonymity.

You choose how you want to use Sopler. You can use it with or without any authentication. A non-authenticated user ensures his privacy and ownership of his list. Alternatively, the authenticated user has the ownership over his items and his list. It’s up to you!

Sopler is free, open source or both ?

Definetly both! The creators of Sopler are governed by the principles and values​of the open source community. Check out Sopler’s source code on GitHub and feel free to fork us. We are open to suggestions, bug or vulnerability reports.



Behind the scenes.

HTML5 / CSS3

Sopler is built using HTML5 & CSS3 supporting the latest web standards. It makes Sopler modern, faster and mobile friendly.

Python / Django

Sopler is built on Django, a high-level Python web framework that helps us make Sopler scalable, fast, and maintainable.

MySQL  

MySQL database server provides high performance and availability, security, scalability and flexibility to the end-users. Sopler uses MySQL to manage the lists.

Apache Server  

Sopler uses the Apache server for performance, scalability and security reasons.

Bootstrap 3  

Sopler's front-end design, based on Twitter's Bootstrap.



Supported technologies.

Sopler is a cross-browser, cross-platform and cross-mobile open web application.
